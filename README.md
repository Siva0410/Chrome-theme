# Chromeの自作テーマ
作成したgoogle chromeのテーマのjsonファイルと画像フォルダ

このテーマを適用するときはディベロッパーモードでパッケージ化されていない拡張機能を読み込むで
このフォルダを選択する

> __注意__ 
拡張機能をパッケージ化でcrxファイルを作成してもchromeでテーマを適用できない
